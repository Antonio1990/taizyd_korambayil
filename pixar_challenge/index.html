Pixar Challenge


This web page displays the results of lighting and shading a scene modeled by Pixar artists for their "Everything & the Kitchen Sink" rendering challenge.
This page is the breakdown for Project 1 in VSFX-502. It shows the process of shading and lighting a scene in Maya using Renderman shaders.

TroubleShooting Random Colors for Crayons.

For the crayons lying on the table I needed to find a quick way to add random colors within a specified range to each object. It didn't make sense to create a different material with the same settigns for each object.

To solve this issue I ended up using the PxrRamp2 Node to create a color Ramp with specified values to assign per-object colors for each Crayon:

Final Thoughts

I found this assignment very difficult, primarily because I am used to working with Real-Time Engines and this workflow of Setting values and waiting for the render to show you the end result felt very cumbersome.
I think my primary problem was my unfamiliarity with Maya and Renderman, I knew what results I wanted to achieve but my lack of knowledge of the software made it very frustrating trying to achieve the results I wanted.

I would have preferred to work on a smaller scene like a still life with only a few objects, so that I could focus on a few shaders and work to refine and perfect them, this would have given me the opportunity to learn more about renderman, the sheer number of objects in this scene was overwhelming for me with no prior experience with Maya or Renderman and it felt like I was just rushing to quickly apply shaders to all the objects.My scene is still quite unfinished in my eyes and will probably work on it some more at my own pace.

I think I will most likely try and work on a smaller still life in my free time to get more familiar with Renderman and all its nodes.
