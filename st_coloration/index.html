ST-Coloration:Maya Patterns


These images show the results of using the Open Shading Language (OSL) to write a variety of shaders for use with RenderMan for Maya. The notes and OSL code accompanying each image explain how each pattern was achieved.



Rectangle

Shader Inputs:

The rectangle shape was set with adjustable parameters for it's positioning in the UV coordinate system, as well as parameters to control it's width and height.I've also exposed parameters to control the colors.

Circle

Shader Inputs:

The circle shape was created by centering the UV coordiante system, and comparing the length of the UV Vector with the radius parameter. The shader has parameters to control the position and radius of the sphere as well as the colors. Controlling the position was only a matter of subtracting the UV coordinates from the x and y inputs.

Crescent

Shader Inputs:

The crescent shape was created by utilizing the same code as the circle, but in this case the circle function is called twice to create two circles, with one slightly offset from the other, then it's simplya matter of subtracting one circle from the other to get the crescent shape.The Shader also has parameters to control position radius and colors. It would also be trivial to create a slider to control the crescent shape to go from full moon to all other shapes.
